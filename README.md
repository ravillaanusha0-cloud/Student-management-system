# Student-management-system
This is a simple Python console application to manage student records.

Features:
- Add new students
- View all students
- Search student by roll number
- Update student marks
- Delete student records
- Save data using file handling (JSON)

Technologies Used:
- Python
- JSON

How to Run:
1. Make sure Python is installed
2. Run the file main.py
3. Follow the menu options on the screen

Learning Outcomes:
- Python dictionaries
- Functions
- File handling
